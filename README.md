# Rick & Morty API App

Este é um aplicativo que consome a **API do Rick & Morty** para exibir os personagens da série, mostrando suas imagens e o status atual: **vivo, morto ou desconhecido**.

## Funcionalidades
- Listagem de personagens com imagem.
- Exibição do status de cada personagem:
  - 🟢 Vivo
  - 🔴 Morto
  - ⚪ Desconhecido

## Tecnologias utilizadas
- Kotlin / Java (Android)
- Retrofit para consumo da API
- RecyclerView para listagem dos personagens

## API
Os dados são obtidos da [Rick and Morty API](https://rickandmortyapi.com/).
